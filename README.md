# Feihong's beets quickstart

## Prerequisites

    pip3 install beets

## Sample config file

```
directory: ~/work/beets-quickstart/music
library: ~/work/beets-quickstart/library.db

import:
  move: yes
```

## Commands

Get location of beets config file:

    beet config -p

Import some music:

    beet import ~/chinese-music-processors/youtube/output
